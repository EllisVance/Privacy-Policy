Async Time-Bridge

Privacy Policy

Effective date: May 19, 2026

Short version: Async Time-Bridge runs locally in your browser. It does not send webpage content, detected timestamps, or your timezone settings to any external server.

1. What Async Time-Bridge Does

Async Time-Bridge is a Chrome extension that detects timezone-based timestamps on webpages, such as “10 AM PST”, “Friday 3pm PT”, or “14:00 GMT+2”, and displays a local conversion tooltip directly on the page.

2. Information We Process

The extension processes the following information locally in your browser:

  • Visible webpage text, only to detect timezone-based timestamp patterns.
  • Detected timestamp strings, only to calculate local and configured team times.
  • User preferences, including enabled state, local timezone, and configured team member names and timezones.

Webpage text and detected timestamps are processed in memory on your device and are not transmitted to us or to any third-party server.

3. Information We Store

Async Time-Bridge uses Chrome’s storage API to save user preferences, including:

  • Your selected local timezone.
  • Whether the extension is enabled.
  • Team member display names and timezone identifiers that you manually add.

These preferences are stored with chrome.storage.sync, which means Chrome may sync them across browsers where you are signed in with the same Chrome account. This sync behavior is provided by Chrome.

4. Information We Do Not Collect

Async Time-Bridge does not collect, sell, share, or transmit:

  • Your browsing history.
  • Full webpage content.
  • Email content, chat messages, task descriptions, or documents.
  • Account credentials, passwords, cookies, or authentication tokens.
  • Personal identifiers such as name, email address, phone number, or payment information.
  • Analytics, advertising identifiers, or tracking data.

5. Network Requests

The extension does not make external network requests for its core functionality. Time conversion is calculated locally using JavaScript libraries bundled inside the extension package.

6. Remote Code

Async Time-Bridge does not execute remote code. All JavaScript files and libraries, including Day.js and its plugins, are included in the extension package and loaded from local extension files.

7. How We Use Information

Information processed or stored by the extension is used only to:

  • Detect timezone-based timestamps on webpages.
  • Display local and team timezone conversions.
  • Remember your extension preferences across sessions.

8. Data Sharing

We do not sell, rent, transfer, or share user data with third parties. The extension does not send user data to our servers because no server-side processing is used.

9. Data Retention and Deletion

Preference data remains in Chrome storage until you change it, remove it in the extension settings, reset browser extension data, or uninstall the extension. Uninstalling the extension removes locally stored extension data according to Chrome’s extension storage behavior.

10. Security

Async Time-Bridge is designed to minimize data exposure by processing webpage text locally and avoiding external transmission. The extension requests only the permissions needed for its functionality.

11. Children’s Privacy

Async Time-Bridge is a productivity tool for professional and personal workflow use. It is not directed to children and does not knowingly collect personal information from children.

12. Changes to This Policy

We may update this Privacy Policy when the extension changes. If material privacy practices change, we will update the effective date and the policy content before publishing the updated extension.

13. Contact

For privacy questions, contact the publisher through the support contact listed on the Chrome Web Store listing for Async Time-Bridge.

14. Chrome Web Store Limited Use Disclosure

The use of information received from Chrome APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

This policy applies only to the Async Time-Bridge Chrome extension.
