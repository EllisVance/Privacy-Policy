Privacy Policy

Review Analytics Shadow Decoder

Effective date: May 19, 2026

Review Analytics Shadow Decoder is a local-first Chrome extension that helps users analyze visible review and comment text from supported websites. The extension is designed to keep analysis inside the user's browser.

No server upload

Review text is processed locally in browser memory.

No external APIs

The extension does not call remote AI, analytics, or processing services.

No data sale

User data is not sold, shared, transferred, or used for advertising.

Single Purpose

The extension analyzes visible review and comment text from supported review pages and converts it into local ASO keyword, phrase, and pain-point insights.

Data Processed Locally

When the user clicks Deconstruct Reviews, the extension may read visible text content from supported pages on Google Play and Product Hunt. This text is processed locally to calculate keyword frequency, phrase frequency, pain-point signals, and CSV export rows.

The extension may also read the active tab URL and title only to detect whether the current page is a supported review source and to include the source URL in a user-initiated CSV export.

Data Not Collected or Transmitted

The extension does not transmit review text, browsing data, CSV content, or analysis results to the developer or to any third party.

  • No remote server is used.
  • No external API is called.
  • No analytics SDK is included.
  • No advertising SDK is included.
  • No account or login is required.

Storage

Review text and analysis results are kept in browser memory while the side panel is open. The extension does not store review text in persistent extension storage.

If the user chooses to export a CSV file, the file is generated locally by the browser and saved only where the user chooses.

Permissions

The extension uses the following Chrome permissions for its single purpose:

  • sidePanel: displays the local analysis interface next to the active page.
  • activeTab: identifies the current active tab after user action.
  • scripting: injects the packaged local content script into supported pages when needed.
  • https://play.google.com/*: reads visible review text on Google Play pages.
  • https://producthunt.com/* and https://*.producthunt.com/*: reads visible comment text on Product Hunt pages.

Remote Code

The extension does not use remote code. All JavaScript, HTML, CSS, and image assets used by the extension are packaged with the extension.

Third-Party Sharing

The extension does not sell, share, transfer, or disclose user data to third parties. It does not use user data for advertising, creditworthiness, lending, or unrelated purposes.

User Control

Analysis runs only when the user clicks Deconstruct Reviews. Users can close the side panel, delete exported CSV files, or remove the extension from Chrome at any time.

Changes to This Policy

This policy may be updated if the extension's functionality or data handling changes. The effective date above will be updated when material changes are made.

Contact

For privacy questions, contact the publisher through the support contact listed on the Chrome Web Store listing.
