Privacy Policy

Eye & Posture Guardian
Effective date: May 19, 2026

Eye & Posture Guardian is a Chrome extension that helps users protect their eyes and posture by showing visual reset prompts, posture reminders, and local preference controls.

The extension is designed to work locally in the browser. It does not collect, sell, transmit, or share personal information.

Information We Collect

Eye & Posture Guardian does not collect personal information, browsing history, page content, keystrokes, analytics data, health data, or account identifiers.

The extension does not create user accounts and does not require login credentials.

Local Browser Storage

The extension stores a small amount of configuration data locally in Chrome using chrome.storage.local. This may include:

  • whether Smart Flow Mode is enabled;
  • whether Posture Reminders are enabled;
  • scheduled reminder timestamps;
  • local reminder preferences for specific sites;
  • temporary local counters such as flow-protection minutes.

This data remains on the user's device. It is not sent to any server controlled by us or by the extension.

Activity Signals

To support Smart Flow Mode, the content script may observe local keyboard, pointer, and scrolling activity events only to estimate whether the user is actively working. The extension does not record typed characters, does not read message contents, and does not transmit activity events.

Chrome Permissions

The extension requests the following Chrome permissions:

  • storage: to save local preferences and reminder timestamps.
  • alarms: to schedule visual reset and posture reminder intervals.
  • idle: to avoid redundant reminders when the user is away or the device is locked.
  • activeTab: to send reminder messages to the currently active tab.

Remote Code

Eye & Posture Guardian does not load or execute remote code. The extension's behavior is provided by packaged extension files.

Data Sharing

We do not sell, rent, transfer, or share user data with third parties. Because the extension does not collect personal data, there is no personal data to disclose to advertisers, analytics providers, or data brokers.

Children's Privacy

The extension is not directed to children under 13. It does not knowingly collect personal information from children.

Changes to This Policy

We may update this Privacy Policy if the extension's functionality or data handling changes. Any material change should be reflected in the published policy before or at the time the updated extension is released.

Contact

For privacy questions, contact us through the support or contact channel listed on the Chrome Web Store listing for Eye & Posture Guardian.

Chrome Web Store Limited Use

The use of information received from Chrome APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
