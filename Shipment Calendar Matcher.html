Shipment Calendar Matcher

隐私政策

生效日期：2026 年 5 月 19 日

摘要：Shipment Calendar Matcher 不收集、不上传、不出售、不共享用户数据。 本扩展仅在用户主动划选网页文本、右键发送文本或手动输入文本时，在浏览器本地进行日期解析和时区转换。

1. 我们是谁

Shipment Calendar Matcher（货运交期多时区对账仪）是一款 Chrome 浏览器扩展，用于帮助用户解析网页、邮件或业务系统中的交期文本， 将日期和时区转换为用户选择的目标时区，并显示倒计时与紧急程度。

2. 我们处理哪些数据

本扩展可能在本地处理以下数据：

  • 用户主动在网页中划选的文本。
  • 用户通过右键菜单发送到扩展的选中文本。
  • 用户手动输入到扩展侧边栏中的交期文本。
  • 解析后的日期、时区、倒计时状态和用户保存的偏好设置。

这些数据仅用于提供日期解析、时区转换、交期倒计时和本地记录展示功能。

3. 我们不收集哪些数据

本扩展不会收集以下数据：

  • 姓名、邮箱、电话号码、地址等身份信息。
  • 账号密码、支付信息或财务信息。
  • 浏览历史记录。
  • 网页中的完整页面内容。
  • 位置数据。
  • 广告标识符或跨站追踪数据。

4. 数据是否会上传到服务器

不会。本扩展不使用外部服务器、远程 API、统计 SDK、追踪像素或广告服务。 日期解析和时区转换全部在用户的浏览器本地完成。

5. 本地存储

为了让用户保留交期记录和偏好设置，本扩展会使用 Chrome 本地存储保存以下内容：

  • 最近解析的交期记录。
  • 目标时区设置。
  • 数字日期格式偏好，例如 MM/DD/YYYY 或 DD/MM/YYYY。

这些数据保存在用户本地浏览器中，不会被发送给开发者或任何第三方。

6. 权限使用说明

本扩展使用以下 Chrome 权限：

  • sidePanel：用于显示扩展侧边栏界面。
  • contextMenus：用于提供“发送到交期对账仪”的右键菜单。
  • storage：用于在本地保存交期记录和用户偏好。
  • activeTab：用于在用户当前操作的网页中处理主动划选的文本。

本扩展请求网页访问能力，是为了在用户访问的网页中监听用户主动划选的文本。 扩展不会扫描、复制或上传完整网页内容。

7. 数据共享与出售

本扩展不会出售、出租、交易、共享或转让用户数据给任何第三方。

8. 第三方服务

本扩展不集成第三方分析服务、广告服务、远程日志服务或云端数据处理服务。 所需的日期处理库随扩展本地打包，不从外部网络加载。

9. 用户控制

用户可以通过以下方式控制本地数据：

  • 在扩展侧边栏中删除单条交期记录。
  • 在扩展侧边栏中清空全部记录。
  • 在 Chrome 扩展管理页面中移除本扩展，以删除扩展相关本地数据。

10. 儿童隐私

本扩展面向跨境供应链、外贸、货代等业务场景，不面向儿童设计。 我们不会故意收集儿童个人信息。

11. 政策更新

如本隐私政策发生重大变更，我们会更新本页面的生效日期，并在必要时通过扩展上架页面或版本说明告知用户。

12. 联系方式

如对本隐私政策或数据处理方式有任何疑问，请通过 Chrome Web Store 中本扩展的开发者联系渠道与我们联系。

Privacy Policy

Effective Date: May 19, 2026

Summary

Shipment Calendar Matcher does not collect, transmit, sell, or share user data. The extension only processes text that the user explicitly selects on a webpage, sends through the context menu, or enters manually into the side panel. Date parsing and timezone conversion are performed locally in the browser.

Data Processed Locally

The extension may locally process:

  • Text explicitly selected by the user on a webpage.
  • Text sent by the user through the extension context menu.
  • Text manually entered by the user into the side panel.
  • Parsed dates, timezone settings, countdown status, and user preferences.

No Data Transmission

The extension does not use external servers, remote APIs, analytics SDKs, tracking pixels, or advertising services. All processing is performed locally.

Local Storage

The extension uses Chrome local storage to save parsed records and user preferences, such as target timezone and numeric date format preference. This information remains in the user's browser and is not sent to the developer or any third party.

Permissions

  • sidePanel: Displays the extension side panel.
  • contextMenus: Provides the context menu action.
  • storage: Saves local records and preferences.
  • activeTab: Supports user-initiated text selection on the active tab.

Data Sharing

The extension does not sell, rent, trade, share, or transfer user data to third parties.

Contact

If you have questions about this Privacy Policy, please contact the developer through the Chrome Web Store listing for Shipment Calendar Matcher.
